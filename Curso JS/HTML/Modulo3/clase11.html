Operadores de Comparación

Tenemos 2 tipos de operadores de Comparación: Relacionales e Igualdad

Relacionales: aquellos que se comparan entre relación, es decir, que se comparan entre dos valores para ver un resultado

  • console.log(a>5) a mayor que 5?
  • console.log(a<5) a menor que 5?
  • console.log(a>=5) a mayor o igual que 5?
  • console.log(a<=5) a menor o igual que 5?

    • Igualdad: Nos sirve para poder sacar resultados comparativos solo entre dos valores para una validación

  • console.log(a==5) // a es exactamente igual que 5
  • console.log(a!=10) // a es distinto a 10?
  • console.log (a== "10") // saldrá true porque solo pregunta si es igual, ignora los tipo de datos
  • console.log(a!="10") //Es distinto el valor de a? Saldrá false porque si es el mismo valor de a
  • console.log(a==="10")// aquí dará false ya que también verifica si el tipo de dato es igual
  • console.log(a!=="10")// Es distinto tipo de dato o valor?
