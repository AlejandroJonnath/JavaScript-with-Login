Tipado Dinámico y Estático

En los lenguajes de programación hay dos tipados: Estático y Dinámico

El estático es aquel que nosotros requerimos colocar el tipo de dato en nuestra variable:

  • String patito= "Patito Feliz"

    • Este tipado está fuertemente estructurado en Java

    En el tipado dinámico nosotros no necesitamos colocar nuestro tipo de dato ya que el programa lo entenderá como en JS

  • variableNumero= 4
  • variableTexto= "texto"
