Datos Primitivos

Para JavaScript nosotros veremos los datos primitivos principales:

  • Int: Es para números enteros (1,2,3,4,5)
  • String: Cadenas de texto ("Hola Mundo")
  • Boolean: Validaciones de verdadero y falso (true, false)
  • Undefined: variable no definida (int calamar;)
  • Null= variable que está definida pero tendrá valor nulo(null)

    • Para poder ver los resultados en la misma página de tu navegador debes abrir inspeccionar, ir a consola y ahí estarán los resultados
